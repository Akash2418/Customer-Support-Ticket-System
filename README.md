# Customer Support Ticket Management System

## Description

This project is a responsive Customer Support Ticket Management System built using **HTML, CSS, and JavaScript**. It allows users to create, view, edit, delete, search, and manage support tickets. All ticket data is stored in the browser using **Local Storage**, so it remains available even after refreshing the page.

## Features

* Dashboard with ticket statistics
* Create new tickets
* View ticket details
* Edit existing tickets
* Delete tickets with confirmation
* Search by Ticket ID, Subject, or Customer Name
* Filter by Status and Priority
* Client-side pagination
* Responsive design for Desktop, Tablet, and Mobile
* Local Storage for data persistence
* Form validation

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Start creating and managing support tickets.
